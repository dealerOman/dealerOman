About DealerOman

DealerOman is a student project for COMP3700 - Introduction to Web Computing.

The website simulates a simple online marketplace for cars, electronics, pets, properties, and freelancer services inside Oman.

Project goals

  • Practice HTML, Bootstrap, and basic JavaScript from the course slides.
  • Use forms, tables, and JavaScript arrays as required in the project instructions.
  • Create a small multi-page website using simple web technologies.

Website structure

Page File Description
Home index.html Main page with categories, search bar, and carousel.
Categories categories.html Dynamic car listings and form to add new cars.
Freelancers freelancers.html Dynamic freelancer table and registration form.
Questionnaire questionnaire.html Simple feedback form about the website.
Calculator calculate.html Simple price calculator using JavaScript.
Fun page funpage.html Small interaction related to cars and ads.
About about.html Information about the project and group members.
Contact contact.html Contact form and email links for the team.
